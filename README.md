# notes-cn-official

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存在，不依赖于特定的后端服务或数据库，适合用于资料存档、知识展示或轻量级内容分发。

所有页面均为独立文件，可直接通过浏览器打开访问，无需额外配置。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面
├── assets/         # 页面关联的资源文件（如样式、脚本、图片等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名即页面标识。
- `assets/`：按需存放各页面引用的公共或私有资源，建议按页面分类组织子目录。

## 页面归档说明

- 所有页面均为静态 HTML，遵循 W3C 标准，兼容主流现代浏览器。
- 页面内容可能包含文本、表格、图表或交互元素，但不会主动收集用户数据或发送网络请求。
- 归档内容定期更新，但不对时效性做特别承诺。如有特定需求，欢迎通过 Issues 提出。

## 维护说明

- 本仓库由维护者不定期更新，不接受未经验证的直接推送。
- 如需提交新页面或修正现有页面，请先创建 Issue 进行讨论。
- 所有贡献需遵循仓库的许可证条款。
